pAIve: Building a Career Intelligence Dashboard with AI

2026  |  4 min read  |  Vibe Coded

This case study documents how I used AI-assisted development (Vibe Coding) to rapidly prototype pAIve — a career intelligence platform concept designed to reduce career discovery time by 50%. Rather than spending weeks on wireframes and handoffs, I partnered with AI to go from problem statement to interactive prototype in a single session.

The point isn't that AI built it for me. The point is that I directed it — shaping the vision, writing the prompts, iterating on the output, and making the creative decisions that turned a blank canvas into a polished product concept.

The Process

Step 1 — The Problem

Career Discovery is Broken

Professionals looking to pivot careers spend an average of 10+ hours researching roles, skill gaps, salary benchmarks, and learning paths — scattered across dozens of tabs, job boards, and outdated blog posts. There's no single source of truth.

I asked: What if an AI-powered platform could consolidate all of this into a single, personalised dashboard — and cut that research time in half?

Core Hypothesis

A well-designed career intelligence dashboard, powered by AI, could reduce career discovery time from 10 hours to under 5 — giving users a clear roadmap instead of information overload.

Step 2 — Sketches & Ideation

High-Fidelity Blueprinting in Figma

Before writing a single line of code, I architected the entire user experience in Figma. I treated the design file as the "Source of Truth" for the AI, defining the exact visual hierarchy, component states, and user flows (Onboarding, Chat, Dashboard) upfront. This allowed me to prompt the AI with specific constraints rather than vague ideas.

I landed on a Bento Grid layout — inspired by Linear, Stripe, and Perplexity — where each card serves a single purpose: Match Score, Salary Range, Skill Gaps, and a Step-by-Step Roadmap.

Step 3 — AI Prompts (Vibe Coding)

Directing AI Like a Creative Director

This is where Vibe Coding comes in. Instead of writing every line of code from scratch, I authored detailed, structured prompts that defined the persona, design system, data architecture, and component structure — then let AI generate the implementation while I directed and iterated.

Here's the actual prompt structure I used:

Prompt — Persona & Design System

"Act as a Senior UI/UX Designer. Role: Implement 'Ready Player One' Aesthetics & Glassmorphism. Global Theme: Apply a Deep Purple Gradient background to the entire app. Replace all 'Solid White' containers with 'Glass' styling: background: rgba(255, 255, 255, 0.03), backdrop-filter: blur(16px), and a subtle white border. Interactive Elements: Add a 'Holographic' glow to the Avatar selection cards (box-shadow: 0 0 20px rgba(120, 80, 255, 0.4)). Ensure all text remains high-contrast white/grey for readability on dark backgrounds."

Prompt — DATA ARCHITECTURE & LOGIC

"Act as a Senior React Developer. Role: Implement Strict JSON Resume Parsing. System Instruction: Analyze the resume text and return ONLY a valid JSON object. Do not include markdown. Strict Schema Definition: fullName: string currentStatus: string (Must be one of: 'Student', 'Fresh Graduate', 'Career Switcher') education: array of objects { institution, degree, yearStart, yearEnd } experience: array of objects { company, role, description, impact_metrics } skills: array of strings Logic Rule: If the user has graduated within the last 2 years and has no full-time role, strictly classify 'currentStatus' as 'Fresh Graduate'."

Prompt — Component Architecture

"Act as a Senior Frontend Engineer. Role: Unify Onboarding Wizard Components. Task 1 (Refactor): Redesign the 'Communication Style' component. Convert the current vertical list into a Grid Card Layout (3 Columns) to match the Avatar Selection step. Task 2 (Visuals): Each card must use the shared Glassmorphism style. content should be centered: Icon (Top), Title (Middle), Description (Bottom). Task 3 (State): Add a 'StepIndicator' badge at the top ({current} of {total}). Ensure the header typography matches the 'Ready Player One' aesthetic defined in the global theme."

Step 4 — Final Output

Live Landing Page & Design System

Deployment: Live on Netlify
Status: Public Beta
Tech Stack: React + Vite
Backend: Supabase
